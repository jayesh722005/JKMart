# 🛒 JKMart

A modern and responsive e-commerce web application built using **React.js** and **Vite**. JKMart provides a seamless online shopping experience with an attractive user interface, product listings, search functionality, shopping cart, and responsive design.

---

## 📌 Project Overview

JKMart is designed to simulate a real-world online shopping platform where users can browse products, search for items, view product details, and manage their shopping cart.

---

## ✨ Features

- 🏠 Beautiful Home Page
- 📦 Product Listing
- 🔍 Search Products
- 🛍️ Shopping Cart
- ❤️ Add to Cart
- 📱 Fully Responsive Design
- ⚡ Fast Performance with Vite
- 🎨 Modern UI

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/jayesh722005/JKMart.git
```

### Navigate to Project

```bash
cd JKMart
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

The application will run on:

```
http://localhost:5173
```
